# 🚀 AI Internship Project Suite

This repository contains 4 AI projects built using NLP and Deep Learning techniques.

## 📌 Projects Included

### 1️⃣ Text Summarization
- Extractive summarization using NLTK
- Abstractive summarization using BART (HuggingFace)

### 2️⃣ Speech Recognition
- Google Speech API (online)
- CMU Sphinx (offline)
- Wav2Vec2 (Deep Learning model)

### 3️⃣ Neural Style Transfer
- Implemented using VGG19
- Uses Content Loss + Style Loss (Gram Matrix)

### 4️⃣ Text Generation
- GPT-2 based text generation
- Supports temperature, top-k, top-p sampling
